# Twitch Chat Local Translation Bot (ポートフォリオ用プロジェクト)

## 1. プロジェクト概要
Twitchの配信チャットを監視し、海外視聴者からの外国語コメント（英語、スペイン語などの欧州言語全般）を検知して、自動で日本語に翻訳してチャットに返信するボットを開発する。

**【最大の特徴・ポートフォリオとしてのアピールポイント】**
DeepLやOpenAIなどの「外部API（有償/回数制限あり）」に一切依存せず、**ローカル環境（自ホスト）で稼働するオープンソースのAIモデル（機械学習モデルまたはローカルLLM）を用いて言語判定と翻訳を完結させる。**
これにより、将来的なAPIコストの問題を解決し、スケーラビリティとWeb ML/ローカルAI技術への深い理解を示す。

## 2. 技術スタック（指定）
*   **言語:** TypeScript / Node.js
*   **Twitch連携:** `tmi.js`
*   **翻訳・言語判定（以下のいずれか、またはAIの推奨を採用）:**
    *   案A: `Transformers.js` (Helsinki-NLPモデル等を使用したNode.js完結のオンデバイス翻訳)
    *   案B: `Ollama` API (ローカルで稼働するLlama3等に対するローカルHTTPリクエスト)
    *   案C: `franc` (言語判定用) + 軽量なローカル翻訳ライブラリ
*   **コード品質:** ESLint, Prettier (またはBiome)

## 3. コア機能要件
1.  **Twitchチャットのリアルタイム監視**
    *   指定したTwitchチャンネルのチャットをリッスンする。
    *   ボット自身の発言や、「!」から始まるコマンドは処理をスキップする。
2.  **完全ローカルでの言語判定（Language Detection）**
    *   外部APIを使わず、メッセージが何語（英語、スペイン語、フランス語など）であるかをプログラム内で判定する。
    *   日本語（JA）と判定された場合は翻訳処理をスキップする。
3.  **完全ローカルでの翻訳機能（Local Translation）**
    *   外部APIを使わず、ローカルのAIモデルを使用して判定された言語から日本語へ翻訳する。
4.  **スパム・誤判定防止フィルタリング（精度向上）**
    *   文字数が極端に少ないコメント（例：3文字以下）や、URL、Twitchの標準エモート名（例："LUL", "PogChamp"）、記号のみのコメントは翻訳・判定対象から除外する。
5.  **UXを考慮したチャット送信（フラグ付与）**
    *   翻訳結果をチャットに送信する際、元の言語の国旗絵文字（例：🇪🇸、🇫🇷、🇺🇸）を付与し、視認性を高める。
    *   出力フォーマット例: `[🇪🇸] @ユーザー名: 翻訳された日本語のテキスト`

## 4. アーキテクチャと設計の制約（品質要件）
ポートフォリオとしてコードを公開するため、以下の設計原則を守ること。
*   **関心の分離 (Separation of Concerns):** 
    *   `index.ts` (エントリーポイント)
    *   `twitch.ts` (Twitchへの接続とイベントハンドリング)
    *   `translator.ts` (ローカルAIモデルのロードと翻訳処理)
    *   `filter.ts` (スパム判定や文字数制限などのバリデーションロジック)
    のように、役割ごとにファイルを分割すること。
*   **エラーハンドリング:** 翻訳モデルのロード失敗時や、予期せぬチャット文字列が来た際にボットがクラッシュしないよう、適切な `try-catch` とログ出力（`console.error`）を実装すること。
*   **ハードコードの禁止:** チャンネル名、ボットのユーザー名、OAuthトークン、翻訳対象の言語リストなどは `.env` ファイルで管理すること。

## 5. 実装ステップ（AIへの指示）
このドキュメントを読み込んだら、以下のステップで段階的に提案・実装を進めてください。

*   **Step 1:** プロジェクトの初期化 (`package.json`, `tsconfig.json` のセットアップ) と、必要なパッケージのインストールコマンドを提示する。ローカル翻訳のアプローチ（Transformers.jsかOllamaか等）について、最適な構成を提案する。
*   **Step 2:** `.env` の設定項目と、Twitchに接続して単純にコンソールにチャットを出力するだけの `tmi.js` のベースコードを実装する。
*   **Step 3:** 外部APIを使わない「言語判定ロジック」と「スパムフィルタリングロジック」を実装し、テストする。
*   **Step 4:** ローカルAIモデルによる「翻訳ロジック」を結合し、Twitchのチャットに国旗付きで結果を返す最終的な処理を完成させる。
